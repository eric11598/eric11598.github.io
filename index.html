EZ Fitness


EZ Fitness created initially for the first team project of the Columbia Engineering Bootcamp. EZ Fitness is a site tailored to creating workout and nutrition plans. Use the basic side to get macronutrient goals and recipes based off of you dietary preferences. Use the advanced side to create custom nutrition plans and workout plans with specific food and excercises.


HTML5, CSS3, Bootstrap, jQuery, Nuritionix API, WGER Workout API

E Dating


E Dating was created as project 2 for the Columbia Engineering Bootcamp. E Dating is a platform that allows you connect with people based on mutual likes, opening up a chat option when you get a match. You login using Facebook and all users are saved on a MongoDB database, as well as their choices on swipes. Mutual likes get matched, and a talk JS chat box opens up.


HTML5, CSS3, Handlebars, jQuery, Node, Express, MongoDB, Heroku, talk JS API, Facebook API

Cashing Winners


Final Project of the Columbia Engineering bootcamp, website was created for an easy way for beginners to get the information they needed to start betting. Provides real time statistics, injury reports, and spreads for all NBA and NFL games going on. Also contains a glossary with common betting terms with a search bar for easy access to anything the beginner may not know. Uses a web scraper to take odds and NFL information off of ESPN.com and uses the MySportsFeeds API to get real time information about the NBA.


HTML5, CSS3, React, jQuery, Cheerio, axios, Node, Express, MongoDB, Heroku, MySportsFeeds API
